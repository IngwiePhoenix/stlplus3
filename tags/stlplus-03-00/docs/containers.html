Introduction

The container library contains abstract data classes which expand on the STL as the name STLplus suggests. These are template classes and require some prior experience of using templates in C++. The problem is that the STL contains only 1-dimensional sequence containers and omits 0-dimensional (single object) and multi-dimensional containers. The STLplus containers provide some of these, specifically a smart pointer, a directed graph, a hash table, an n-ary tree, and a 2-dimensional matrix. It also extends the concept of the std::pair to 3 and 4 element collections.

Dependencies

The portability library has no dependencies and so can be used as a stand-alone library.

Contents

The components provided are:

  • smart_ptr.hpp: Smart Pointer
  • digraph.hpp: Directed Graph
  • hash.hpp: Hash Table
  • ntree.hpp: N-ary Tree
  • matrix.hpp: 2-Dimensional Matrix
  • triple.hpp: Triple
  • foursome.hpp: Foursome
  • safe_iterator.hpp: Safe Iterators
